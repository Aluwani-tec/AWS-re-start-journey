# Database Module

This folder contains my work for the **Database module** completed as part of the
AWS re/Start Programme.

The lab covered both **managed cloud databases** and **hands-on SQL practice**
to demonstrate understanding of relational and NoSQL database concepts.

## Topics Covered
- Amazon RDS (Relational Database Service)
- DynamoDB (NoSQL database)
- SQL operations using MariaDB/MySQL on EC2
- Database creation and table design
- Querying data and joining tables

## Contents
- `notes.md` – Detailed step-by-step lab notes with explanations and screenshots
- `images/` – Screenshots used as proof of completed tasks

This work demonstrates practical database skills aligned with AWS cloud best practices.
