This folder contains hands-on labs completed as part of the **AWS re/Start Programme**.  
Each lab focuses on practical application of cloud and IT concepts using real AWS services and Linux environments.

The labs are organized by domain to clearly demonstrate skill progression across core cloud competencies.

---

## Application Labs
The **Application** labs focus on deploying and validating application workloads on AWS infrastructure.

Key areas covered include:
- Hosting applications on Amazon EC2
- Validating application accessibility via public IPs
- Understanding how applications interact with underlying infrastructure
- Observing application behavior in scalable environments

These labs emphasize the relationship between infrastructure and application availability.

---

## Compute Labs
The **Compute** labs focus on Amazon EC2 and related services that provide on-demand compute capacity.

Key areas covered include:
- Launching and managing EC2 instances
- Creating Amazon Machine Images (AMIs)
- Using launch templates
- Implementing Auto Scaling groups
- Understanding elasticity, fault tolerance, and high availability

These labs reinforce how AWS compute services scale and self-heal to meet demand.

---

## Database Labs
The **Database** labs introduce managed database services and basic data operations in AWS.

Key areas covered include:
- Understanding relational database concepts
- Working with managed database services
- Connecting applications to databases
- Performing basic queries and data validation

These labs build foundational knowledge of data persistence in cloud environments.

---

## Linux Labs
The **Linux** labs focus on operating system fundamentals required for cloud and IT roles.

Key areas covered include:
- Navigating the Linux file system
- Managing users and permissions
- Running and troubleshooting commands
- Understanding process and service management

These labs strengthen command-line skills essential for managing cloud resources.

---

## Networking Labs
The **Networking** labs focus on AWS networking fundamentals and virtual network design.

Key areas covered include:
- Amazon VPC concepts
- CIDR blocks and IP addressing
- Subnets and network segmentation
- DNS configuration
- Public networking principles

These labs establish the foundation for secure and scalable cloud architectures.
